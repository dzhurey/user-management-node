|  |  +-- utils // utility function for global purpose

## Requirement

1. Nodejs version > 12

2. MySQL

## Installation

### 1. Clone this repository and enter to directory

### 2. Setup dev server:

1. Enter `server` directory

2. Copy file `env.example` and rename it to `.env` then adjust its value

3. Install dependency

```
npm install
```

4. Run Dev Server

```
npm run start:dev
```
